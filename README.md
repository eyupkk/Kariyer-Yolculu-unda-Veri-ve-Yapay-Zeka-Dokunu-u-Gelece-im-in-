# Kariyer-Yolculugunda-Veri-ve-Yapay-Zeka-Dokunusu-Geleceiimcin-
"Geleceğim İçin", kullanıcıların kişilik özelliklerini, kriz yönetimi reflekslerini ve ilgi alanlarını analiz eden 15 soruluk gelişmiş bir algoritma üzerine kurulu.Bu algoritma sonucunda elde edilen veriler; Tıp’tan Yazılım’a, Hukuk’tan Sanat’a kadar geniş bir yelpazede değerlendirilerek bir rapor olarak sunuyor.
